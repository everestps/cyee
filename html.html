HTML and CSS

The basics, plus tips, tricks, and what not to do

HTML, the abbreviation for "Hypertext Markup Language", is a widely used programming language for designing webpages. Though HTML content is fairly static - what appears on the screen will not change much - the language"s ability to integrate other programs written in such languages as CSS, Javascript, and PHP allows for very dynamic website design.

Of these, CSS, or "Cascading Style Sheets", is nearly inseparable from HTML - in fact, it is often considered bad form to use HTML without it. CSS allows website designers to stylize their pages with settings as simple as fonts to as complicated as animations (though the latter is normally left to Javascript). Most of these features can be accomplished with HTML on its own, but CSS also provides a separation of content from style, which makes the code easier to read and change when necessary.

HTML uses a tagging system to define the structure of its sites; the content is written inside each tag. A tag, or more accurately, tag pair, consists of an opening and closing tag, with content in-between the two. For example, the basic paragraph tag appears as <p> paragraph text </p>. In only a select few cases, a tag is permitted to act as both an opening tag and a closing tag. The most common example of this is the image tag, which is written <img src="image redirect" />.

Some common tags include the following (view full documentation):

Tags Attributes Use
<body> visual code </body> none Contains the body of code defining the visual page in the HTML program.
<a href=""> linked text or element </a> href Places a link to the destination designated by href on the contents, which can redirect by url or by directory.
<table> table rows and table cells </table> none Creates a table, which can then be extended by tr for table rows and td for table cells.
<link href="" rel="" type="" /> href rel type Links a site to a document to be used for the site; href defines the document, rel defines how the document will be used ("stylesheet" for a CSS document), and type relates the format of the document ("text/css" for a CSS document).

In this table, a new term, "attributes", appears. Attributes serve as additional information about elements that HTML can use. href, for example, is used in multiple tags to define a url direct. There are more HTML attributes to each of the tags listed above, but the ones listed are the most important and most common. Importantly, <table> is listed as having no attributes due to the functioning of CSS.

CSS, as before noted, is designed to manage all of the styling of a webpage. Before CSS became widely used, HTML attributes defined the styling; now however, styling is expected to occur in CSS - the fifth iteration of HTML (HTML5) even went so far as to remove many attributes from its visual tags for this reason. With CSS3, CSS is also able to handle some animations, which used to require Javascript.
